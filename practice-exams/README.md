# SP Study Guide — Practice Exams

Internet Expert — Service Provider / Expert-level SP practice questions.

## Format

Each question is scenario-based, config-analysis, or troubleshooting. Multiple-choice with a single best answer unless noted. Answers and explanations follow each section.

**Difficulty:** Questions are calibrated to Internet Expert written/lab level. Some are expert-level equivalent. Expect to struggle — that's the point.

## Scoring

| Score | Level |
|-------|-------|
| 90–100% | Ready for the exam |
| 75–89% | Strong — review missed topics |
| 60–74% | Good foundation — targeted study needed |
| Below 60% | Build the concepts, then return |

## Tips

- Don't look at answers until you commit to an answer
- For wrong answers: read the explanation, find the relevant section, re-read it
- Config questions: mentally trace the control plane and data plane
- Troubleshooting: eliminate before selecting — wrong answers are often "almost right"
